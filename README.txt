Este proyecto es el resultado del curso de python introductorio en la facultad de ingeniería
de la Universidad de Antioquia.
Se trata de una aplicacion de python usando tkinter, para manejar las finanzas personales.
En el curso, muy básico, se centró en los fundamentos de python, las estructuras de datos y 
la creacion y manejo de funciones.
Como ejercicio individual debimos aprender el manejo de tkinter y matplotlib.

La app presenta una interfaz con tres pestañas:
1) para el registro de movimientos.
2) para el analisis mensual
3) para el resumen anual mediante gráficos.

Toda la app funciona basada en la lectura y modificacion de un archivo de texto, que hace las veces
de "base de datos" de los registros de gastos e ingresos monetarios de una persona. Esto
obviamente debe adaptarse a una base de datos real, pero como no era el objetivo ni el alcance
del curso, se escibió sobre esta base.

Espero sea de utilidad para alguien!!
